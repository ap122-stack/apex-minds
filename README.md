# Apex Minds - Premium AP Tutoring Platform

Elite tutoring website with Stripe-integrated payment checkout, tutor marketplace, and session booking system.

## Features
- Tutor marketplace with real-time availability
- Stripe subscription checkout for multiple packages
- Session booking and management
- Premium UI with smooth animations
- Test Prep, Standard, and Advanced Placement tiers

## Deployment
Deployed on Vercel with automatic builds from git pushes.
